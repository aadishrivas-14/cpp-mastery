# Week 1: Foundations & Core Concepts

## Daily Schedule
- **Day 1-2**: C++ Basics, Control Flow, Functions
- **Day 3-4**: OOP, Classes, Inheritance, Polymorphism
- **Day 5-6**: Memory Management, RAII, Smart Pointers
- **Day 7**: Project Integration & Review

## Learning Objectives
- Master C++ syntax and core language features
- Implement object-oriented design patterns
- Understand memory management and RAII principles
- Build production-ready classes with proper resource management

## Key Topics
- Variables, operators, control structures
- Functions, overloading, templates basics
- Classes, constructors, destructors
- Inheritance, virtual functions, polymorphism
- Pointers, references, dynamic memory
- Smart pointers, RAII, rule of 3/5/0
- Exception handling basics

- Memory leak detection
