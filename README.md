# C++ Mastery Intensive Program

A fast-paced 4-week intensive C++ program covering everything from basics to advanced topics with data structures, algorithms, and real-world projects.

## Program Structure

### Week 1: Foundations & Core Concepts
- **[README](week-1/)**: C++ Basics, OOP, Memory Management
- **[Theory](week-1/theory/)**: Detailed concepts and examples
- **[Projects](week-1/projects/)**: Banking System, Smart Pointers

### Week 2: STL, Templates & Data Structures
- **[README](week-2/)**: STL, Templates, Custom Data Structures
- **[Theory](week-2/theory/)**: Advanced concepts and patterns
- **[Projects](week-2/projects/)**: Generic Library, Algorithm Visualizer

### Week 3: Advanced C++ & Algorithms
- **[README](week-3/)**: Concurrency, Modern C++, Algorithm Design
- **[Theory](week-3/theory/)**: Performance and optimization
- **[Projects](week-3/projects/)**: Multi-threaded Engine, Trading System

### Week 4: Specialization & Portfolio
- **[README](week-4/)**: Networking, Databases, Final Integration
- **[Theory](week-4/theory/)**: System design and architecture
- **[Projects](week-4/projects/)**: Distributed System, Portfolio Project

## Getting Started

1. Ensure you have a C++ compiler installed (GCC, Clang, or MSVC)
2. Set up your development environment (VS Code, CLion, or preferred IDE)
3. Start with Week 1 and progress sequentially
4. Complete all exercises and projects before moving to the next week
